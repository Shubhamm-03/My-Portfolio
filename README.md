This repository hosts the source code for my personal portfolio website — a place to
showcase who I am, what I'm learning, and what I've built so far.

I'm a B.Tech CSE (AI) student who completed a full web development course and now builds
front-end projects for fun. This site itself is one of those projects: a dark, minimal
portfolio built entirely with **vanilla HTML, CSS, and JavaScript** — no frameworks, no
libraries, just fundamentals.

**What's inside:**
- Responsive single-page layout (About, Skills, Projects, Contact)
- Custom animations and scroll effects
- A showcase of ongoing project work, including a 3D sneaker e-commerce concept
